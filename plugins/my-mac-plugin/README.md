# My Mac Plugin

macOS Produktivitäts-Plugin für Claude Code mit nützlichen Entwickler-Befehlen.

## Installation

```bash
/plugin install my-mac-plugin@dimetrics-marketplace
```

## Befehle

| Command | Beschreibung |
|---------|--------------|
| `/system-info` | Zeigt macOS System-Informationen |
| `/cleanup` | Räumt Caches und temporäre Dateien auf |
| `/ports` | Zeigt belegte Ports und Prozesse |

## Beispiele

```bash
# System-Status prüfen
/system-info

# Speicherplatz freigeben
/cleanup

# Port 3000 prüfen
/ports 3000
```
